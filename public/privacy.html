English

Overview

channelist exports your public YouTube subscriptions into text formats and lets you share them as an image. It does not require any login.

Data we do not store

We do not store your handle, channel ID, subscription list, or any generated output. Your input is passed through our Cloudflare Pages Function only to call the YouTube Data API and return the result to your browser. Nothing is written to logs or a database.

What is processed

  • The handle / channel URL you enter is sent to our server function, which calls the YouTube Data API v3 using a server-side API key. The key is never exposed to your browser.
  • Channel icons may be fetched through an image proxy (/api/thumbnail) restricted to YouTube image hosts, solely to draw the share image. Images are not retained.
  • The share image is generated entirely in your browser (Canvas). It is never uploaded to our servers.

Local storage

The only value stored in your browser is your language preference (JA/EN). No other data is placed in local storage or cookies.

Third parties

Requests are served through Cloudflare and the YouTube Data API (Google). Their handling of network requests is governed by their own policies.

Your subscriptions visibility

To use this app you temporarily turn off "Keep all my subscriptions private" in YouTube. While off, your subscription list is publicly viewable by anyone. Please turn it back on after exporting.

日本語

概要

channelist は、公開設定にしたYouTubeの登録チャンネルをテキスト形式で書き出し、画像として共有できるツールです。ログインは不要です。

保存しないデータ

ハンドル名、チャンネルID、登録チャンネル一覧、生成した出力を一切保存しません。入力はYouTube Data API を呼び出して結果をあなたのブラウザに返すためだけに Cloudflare Pages Function を通過し、ログやデータベースには書き込まれません。

処理される内容

  • 入力したハンドル/チャンネルURLはサーバー関数へ送信され、サーバー側のAPIキーで YouTube Data API v3 を呼び出します。キーがブラウザに露出することはありません。
  • チャンネルアイコンは、共有画像を描画するためだけに、YouTube画像ホストに限定した画像プロキシ(/api/thumbnail)経由で取得される場合があります。画像は保持しません。
  • 共有画像はすべてブラウザ内(Canvas)で生成され、サーバーへアップロードされることはありません。

ローカルストレージ

ブラウザに保存されるのは言語設定(JA/EN)のみです。それ以外のデータをローカルストレージやCookieに置くことはありません。

第三者

通信は Cloudflare と YouTube Data API(Google)を経由します。ネットワークリクエストの取り扱いは各社のポリシーに従います。

登録チャンネルの公開状態について

本アプリを使う際は、YouTubeの「チャンネル登録をすべて非公開にする」を一時的にオフにします。オフの間、あなたの登録チャンネル一覧は誰でも閲覧可能になります。書き出し後は必ず設定を元に戻してください。
